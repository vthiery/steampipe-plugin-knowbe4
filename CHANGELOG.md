## v0.0.1 [2026-03-13]

_What's new?_

- Initial release of the plugin with 13 tables:
  - [knowbe4_account](docs/tables/knowbe4_account.md) — KnowBe4 account details including subscription level and current risk score
  - [knowbe4_account_risk_score_history](docs/tables/knowbe4_account_risk_score_history.md) — historical risk score records for the account
  - [knowbe4_group](docs/tables/knowbe4_group.md) — user groups defined in the account
  - [knowbe4_group_risk_score_history](docs/tables/knowbe4_group_risk_score_history.md) — historical risk score records for each group
  - [knowbe4_phishing_campaign](docs/tables/knowbe4_phishing_campaign.md) — phishing campaigns configured in the account
  - [knowbe4_phishing_recipient](docs/tables/knowbe4_phishing_recipient.md) — phishing security test recipient results
  - [knowbe4_phishing_security_test](docs/tables/knowbe4_phishing_security_test.md) — phishing security tests run within phishing campaigns
  - [knowbe4_training_campaign](docs/tables/knowbe4_training_campaign.md) — training campaigns configured in the account
  - [knowbe4_training_enrollment](docs/tables/knowbe4_training_enrollment.md) — training enrollment records for users
  - [knowbe4_training_policy](docs/tables/knowbe4_training_policy.md) — uploaded compliance policies
  - [knowbe4_training_store_purchase](docs/tables/knowbe4_training_store_purchase.md) — training modules purchased from the KnowBe4 content store
  - [knowbe4_user](docs/tables/knowbe4_user.md) — users registered in the account
  - [knowbe4_user_risk_score_history](docs/tables/knowbe4_user_risk_score_history.md) — historical risk score records for each user
